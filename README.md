# User Dashboard

לוח מחוונים למשתמש - ממשק מודרני ומהיר שנוצר מפיגמה.

הפרויקט המקורי זמין ב: https://www.figma.com/design/K48aqnysSi17AxtFlr5hDf/User-Dashboard

## 🚀 טכנולוגיות

- React 18
- Vite
- TypeScript
- Tailwind CSS
- shadcn/ui
- Radix UI

## 📦 התקנה

```bash
npm install
```

## 🛠️ פיתוח מקומי

```bash
npm run dev
```

הפרויקט יפתח בכתובת: `http://localhost:5173`

## 🏗️ בנייה לפרודקשן

```bash
npm run build
```

הקבצים הסטטיים ייווצרו בתיקיית `dist/`

## 🌐 Deploy ל-Vercel

הפרויקט מוכן ל-deploy אוטומטי ב-Vercel:

1. העלה את הקוד לגיטהאב
2. התחבר ל-Vercel והתחבר את ה-repository
3. Vercel יזהה אוטומטית שזה פרויקט Vite ויבנה אותו
4. הלינק לא
